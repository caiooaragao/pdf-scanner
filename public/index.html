PDF Scanner

Tudo roda no seu navegador: escolha um ou mais PDFs, aplique o efeito de digitalização (cinza + leve desfoque + JPEG) e baixe o resultado. Não é necessário servidor local.

Arraste PDFs aqui ou

Apenas PDF · vários arquivos de uma vez · requer internet para carregar as bibliotecas

    Arquivo local: duplo clique neste HTML (Chrome ou Edge). O processamento é feito no navegador (precisa de internet para o CDN).

    Opcional (só no seu PC): npm run web e abra http://localhost:3847 — mesma tela com processamento no Node (Sharp, igual ao npm start). Em sites públicos (ex. GitHub Pages) só vale o processamento no navegador.
